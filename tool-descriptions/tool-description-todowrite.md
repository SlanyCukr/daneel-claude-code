Create and manage a structured task list for complex multi-step tasks. Use when a task requires 3+ distinct steps. Skip for trivial single-step tasks.

Task states: pending, in_progress, completed. Keep exactly one task in_progress at a time. Mark tasks complete immediately after finishing. Provide both content ("Fix auth bug") and activeForm ("Fixing auth bug") for each task.
